1. 컴퓨터 사칙연산 (덧셈만 하는 이유)
= 디지털 논리 설계에서 AND Gate, OR Gate 찾아보기
= 하드웨어 효율성을 위해서 덧셈으로 표현
  -> 하드웨어 효율성이란 무엇인

= ALU (연산 블럭)
  -> 구현 시 구성하는 logic gate 들은 boolean arithmetic 을 따름
  -> 덧셈은 회로로 구현하기 간단한 반면, 뺄셈은 복잡함
  -> 컴퓨터는 가감산기, 그러나 감산보다 가산이 빠르므로, 가산 만으로 작동되게 만들려고 한게 현재의 컴퓨터

= 회로가 복잡하다
   -> 데이터가 처리되기 위해 거쳐야 하는 logic gate들이 많다
   -> 처리 시간이 오래 걸린다
